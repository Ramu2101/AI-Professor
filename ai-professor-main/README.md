# AI Professor (Streamlit Edition)

AI Professor is a professional Streamlit-based AI learning platform that turns a topic into:
- Structured explanation
- Real-world applications
- Concept diagram
- Guided roadmap
- Suggested projects and interview preparation
- Relevant YouTube learning video

## Tech Stack
- Streamlit (frontend + deployment)
- Python backend logic
- YouTube Data API v3
- OpenAI or Gemini for content generation
- Graphviz fallback for diagrams
- `st.cache_data` / `st.cache_resource` for performance

## Environment Variables
Create `.env` from `.env.example` and set:
- `YOUTUBE_API_KEY`
- `OPENAI_API_KEY` (optional if using Gemini)
- `GEMINI_API_KEY` or `GOOGLE_API_KEY` (optional if using OpenAI)

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud Deployment
1. Push this project to GitHub.
2. In Streamlit Cloud, click **New app**.
3. Select repo + branch.
4. Set main file path to `app.py`.
5. In app **Settings -> Secrets**, add:
   - `YOUTUBE_API_KEY`
   - `OPENAI_API_KEY` and/or `GEMINI_API_KEY`
6. Deploy.

## Notes
- The app caches YouTube and LLM outputs for 1 hour to reduce cost and latency.
- If YouTube quota is exceeded, the app shows a clean warning and continues.
- Mermaid source is generated by LLM; Graphviz fallback is always shown for reliability.
